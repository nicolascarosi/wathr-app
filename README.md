### 2. Setup guide

#### 2.1 Requirements:

 - GIT
 - NodeJs
 - npm

#### 2.2 Setup:

1. Clone the repository.

2. Install the dependencies by executing the following command on the project folder:

```
npm install
```
	
3. Run the following command to start the application:

```
npm start
```
